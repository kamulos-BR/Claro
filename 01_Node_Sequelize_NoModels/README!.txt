Case sobre esse projeto:

Projeto realizado e em uso oficial para uma compania de venda de seguros
A ideia do cliente é ter um backend organizado que trabalhe com NODEjs + MySQL e AWS
A cada entrada de dados via esses canais: 
CHAT, FORMULARIO NO SITE e EMAIL
uma lambda function para cada tipo de canal irá analisar e tabular os dados do cliente 
e salvar no banco de dados, para ser enviado ao CRM do cliente.
